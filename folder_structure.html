Note: All abbreviations will be explained below the following table.

File/Folder Short description
icons-mk Contains all project files and folders
_svg_total Contains all icons in one file (icons-mk.svg)
css Contains a file with formatting rules (style.css)
img Contains images with pixel graphics
js Contains JavaScript files for dynamic purposes
lib Contains external libraries and info files
logo Contains the logo of icons-mk
svg Contains all icons as vector graphics in separated files
_footer.html Footer component (dynamically loaded via JavaScript file)
folder_structure.html Content component (dynamically loaded via JavaScript file)
index.html Contains all dependencies to other files as well as displays the content of icons-mk
web_usage.html Content component (dynamically loaded via JavaScript file)

css (Cascading Style Sheets), html (Hypertext Markup Language), img (Images), js (JavaScript), lib (Libraries), mk (Matthias Kuhnel), svg (Scalable Vector Graphics)
