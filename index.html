Welcome to icons-mk! :)

icons-mk is an icon-collection created by Matthias Kuhnel. Furthermore, it is published as a part of reuseIT. For more information, please visit The ReUse Project's website (https://reuseproject.org).

Web usage

You can use icons-mk for your website, mobile application or software.

Copy the svg folder to your project. Take the pre-defined code per icon from below and put it in your code. By opening the accordion, you will see all icons.

Your possibilities

  • Put icons-mk directly in to your project
  • Edit icons-mk via SVG code*
  • Edit icons-mk with CSS/jQuery (use a search engine for further information)
  • Edit icons-mk with open-source-software (e.g. with Inkscape)
  • Add animations to the icons-mk (e.g. with Synfig Studio)
  • Transform SVG of icons-mk into icon font (e.g. with icomoon)
  • ...

* Edit SVG code

  1. Open an icon file (e.g. alert.svg) or the total file (icons-mk.svg) with an text editor (e.g. Notepad++ or Visual Studio Code)
  2. Type in #000000 into find field (Ctrl+F)
  3. Type in your color in hex code (e.g. #6a9225) into replace field (Ctrl+H)
  4. Click the option "Replace all"
  5. Save the file (Ctrl+S)
  6. Excellent! You have changed the color of one or all icons

Hint: If you want to change all icons in the separated files (folder: svg), then first open all icon files in your text editor. Choose the options "Find in files" (Ctrl+Shift+F) and "Replace in files" (Ctrl+Shift+H) to type in your hex codes (#000000 and #6a9225). Then click the option "Replace all", save it and you are done :).
